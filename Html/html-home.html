Level 1

Html কী ?

HTML (HyperText Markup Language) হলো একটি মার্কআপ ল্যাঙ্গুয়েজ, যা ওয়েবপেজ তৈরি এবং কাঠামো নির্ধারণ করতে ব্যবহৃত হয়। এটি ওয়েব ডকুমেন্টের বিষয়বস্তু যেমন টেক্সট, ছবি, লিঙ্ক, ভিডিও, টেবিল, ফর্ম ইত্যাদি প্রদর্শন করার জন্য ব্যবহৃত হয়।

File Extension

FileName.html
Html ব্যবহার করার জন্য আপনাকে প্রথমে একটি নিজের মন মতো ফাইল এর নাম দিতে হবে , তারপর অবশ্যই .htm / .html File Extension টি দিতে হবে ।




Html Common Code

Html করতে গেলে আপনাকে এই Code টি বার বার লিখতে হতে পারে , তাই মনে রাখার চেষ্টা করুন ।




HTML Tag কী ?

HTML Tag হলো Html এর একটি মৌলিক উপাদান, যা ওয়েব পেজের স্ট্রাকচার গঠন করতে ব্যবহৃত হয়। প্রতিটি HTML Tag এর একটি নির্দিষ্ট কাজ বা নির্দেশনা থাকে । ট্যাগগুলো সাধারণত বন্ধনী দিয়ে ঘেরা থাকে-

< > এটি দ্বারা ট্যাগ এর শুরু বুঝায়
< / > এটি দ্বারা ট্যাগ এর শেষ বুঝায়




Html Comments

HTML কমেন্ট এমন একটি উপাদান, যা কোডের অংশকে ব্যাখ্যা করতে বা নোট লিখতে ব্যবহৃত হয়। এটি ব্রাউজারে দৃশ্যমান হয় না এবং পেজের আউটপুটে কোনো প্রভাব ফেলে না। সাধারণত ডেভেলপাররা কোড বোঝার সুবিধার জন্য বা সাময়িকভাবে কোড এড়িয়ে যাওয়ার জন্য এটি ব্যবহার করেন। নিচে একটি এর উদাহরণ তুলে ধরা হলো --
